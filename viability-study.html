Estudio de Viabilidad del Proyecto

Creado por: Equipo Fundador Nexus

Fecha: 21 de Octubre de 2025

1. INTRODUCCIÓN Y OBJETIVOS

Nexus es una aplicación móvil y web que combina lo mejor de Chollometro y Wallapop, ofreciendo una plataforma donde los usuarios pueden descubrir ofertas destacadas, productos de segunda mano y promociones de empresas en un solo lugar. El objetivo principal es crear un espacio interactivo y visualmente atractivo para buscar productos, publicar anuncios, influir en la visibilidad con “likes” y usar un sistema de envíos gestionado por Nexus Go.

2. ALCANCE PRELIMINAR

El proyecto busca desarrollar una aplicación funcional para Android (y posteriormente iOS) con: inicio con ofertas, apartado de segunda mano, sección de destacados por likes, buscador global, perfiles de usuario y el módulo de envíos Nexus Go. Inicialmente, se excluyen pasarelas de pago externas y funcionalidades avanzadas de chat.

3. VIABILIDAD TÉCNICA

Tecnologías: Se utilizará Angular y Android Studio (Java) para el frontend, y Java Spring Boot, NodeJS y PostgreSQL para el backend. El proyecto es compatible con dispositivos Android y web, con integración futura para iOS. Se requiere un equipo de 3 desarrolladores.

4. VIABILIDAD ECONÓMICA

Costes estimados iniciales: Desarrollo y diseño (6.000–8.000 €), Servidores y almacenamiento (500 €/año), y Marketing inicial (1.000 €). La financiación provendrá de fondos propios o microinversores. La monetización futura se contempla mediante comisiones de Nexus Go y publicidad destacada para empresas.

5. VIABILIDAD OPERATIVA

Los usuarios disfrutarán de una experiencia simple y moderna con incentivos sociales. Las empresas podrán promocionar productos mediante anuncios. Internamente, se requerirá un panel de administración para controlar publicaciones, reportes y estadísticas de uso.

6. VIABILIDAD LEGAL Y NORMATIVA

Se asegurará el cumplimiento del RGPD, se establecerán términos y condiciones claros, se implementará verificación de identidad para la seguridad en la segunda mano y se protegerá la propiedad intelectual de los contenidos subidos.

7. RIESGOS INICIALES

Riesgo Descripción Medida de mitigación
Competencia alta Existen apps similares como Wallapop o Chollometro. Diferenciarse por la integración de ambas funciones y el sistema de likes.
Falta de usuarios iniciales Riesgo de baja adopción al principio. Campañas de lanzamiento y bonificaciones por registros.
Publicaciones fraudulentas Riesgo de productos falsos o estafas. Moderación y verificación con IA y reportes de usuarios.
Costes operativos Mantenimiento de servidores y comisiones. Escalado progresivo según el número de usuarios.

8. CONCLUSIÓN Y RECOMENDACIÓN

El proyecto Nexus es viable técnica y económicamente, con un enfoque innovador. Se recomienda continuar con el desarrollo del prototipo funcional y realizar una prueba piloto para validar la aceptación del mercado.
